# CineMatch: Movie Recommendation System 🎬

CineMatch is a machine learning-powered movie recommendation engine. The application processes a dataset of 45,000+ movies to deliver content-based recommendations using vector similarity modeling.

🔗 **[Live Web Application](https://cinematch-fv58azmq8qpzys7mppmnwr.streamlit.app/)**

---

## 🛠️ Tech Stack
* **Language:** Python
* **ML/Data:** Scikit-Learn, Pandas, NumPy
* **Frontend:** Streamlit
* **API Integration:** TMDb REST API
